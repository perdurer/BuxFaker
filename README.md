## üöÄ How It Works

### 1. Setting the Balance
Click the extension icon in your toolbar. Enter your desired amount in the text field. The balance is automatically formatted (e.g., `1.2M+` or `500K+`) on the Roblox interface for total realism.

### 2. Protected Navigation
The extension dynamically injects scripts into Roblox pages:
* The navigation bar balance is replaced site-wide.
* The Robux dropdown menu is modified to look authentic.
* **Safety Intercept**: If a user clicks on "Transactions," the extension intercepts the request and force-loads a 404 page, simulating a Roblox system glitch rather than revealing your actual balance.

### 3. Visual Polish
Hover over the logo in the popup to trigger a **smooth rotation animation** followed by a **re-assembly effect**, giving the extension a high-end, polished feel.
